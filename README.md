# ResoGifImporter

A [ResoniteModLoader](https://github.com/resonite-modding-group/ResoniteModLoader) mod for [Resonite](https://resonite.com/) <br>
Crude port of GifImporter for NeosVR

<br>

## Installation
1. Install [ResoniteModLoader](https://github.com/resonite-modding-group/ResoniteModLoader).
2. Place [ResoGifImporter.dll](https://github.com/zahndy/ResoGifImporter/releases/latest/download/ResoGifImporter.dll) into your `rml_mods` folder. This folder should be at `C:\Program Files (x86)\Steam\steamapps\common\Resonite\rml_mods` for a default install. You can create it if it's missing, or if you launch the game once with ResoniteModLoader installed it will create the folder for you.
3. Place [SixLabors.ImageSharp.dll](https://github.com/zahndy/ResoGifImporter/releases/latest/download/SixLabors.ImageSharp.dll) into either your `rml_libs` or `rml_mods` folder.
4. Start the game. If you want to verify that the mod is working you can check your Resonite logs.
